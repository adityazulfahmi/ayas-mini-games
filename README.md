# 🌸 Aya's Mini Games

A collection of fun, calming mini games designed with a cheerful baby girl pastel theme.
All games are single-file HTML — just open in any browser, no installation needed.

## 🎮 Games

| Game | Description |
|------|-------------|
| [🃏 Flip & Matching](./ayas-flip-matching/) | Flip cards and match pairs of fruits & animals. Choose from 2×2 up to 6×6 grid. |
| [⭕ Tic-Tac-Toe](./ayas-tictactoe/) | Classic 2-player Tic-Tac-Toe with 🌸 vs ⭐. Tracks scores across rounds. |

## 🎨 Design

- Soft lavender-to-pink gradient backgrounds
- Pastel purples, pinks, and mints
- Playful *Fredoka One* font
- Responsive — works on mobile and desktop
- Web Audio API sounds (no audio files needed)
